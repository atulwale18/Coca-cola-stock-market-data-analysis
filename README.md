# Coca-Cola Stock Market Data Analysis using Python

This project performs Exploratory Data Analysis (EDA) on Coca-Cola stock market data
to understand price trends, trading volume, and market behavior.

## Datasets Used
- Stock History: Used for analyzing price, volume, and trends
- Stock Info: Used to understand company and business context

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Datetime and timezone handling
- Closing price trend analysis
- Volume analysis
- Moving averages
- Correlation analysis

## Key Insights
- Coca-Cola stock shows a long-term trend with short-term fluctuations.
- Moving averages help identify trend direction.
- Volume spikes indicate high market activity periods.

## Conclusion
This analysis provides useful insights into Coca-Cola stock performance
and helps understand real-world financial data behavior.
